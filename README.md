Magna Badminton Alert

What it does
- Monitors up to 4 PerfectMind class detail pages
- Checks on a fixed interval (minutes)
- Sends a notification when a monitored class transitions from not open to open

LibreWolf and Firefox test install
1. Unzip the folder
2. Open about:debugging#/runtime/this-firefox
3. Click Load Temporary Add-on
4. Select manifest.json

Notes
- Interval is in minutes. Sub-minute intervals are not reliable with alarms.
- Use the class detail page URL (Clients/BookMe4LandingPages/Class) for best results.
